## 微博爬虫规范

#### By ihciah & Skyduy

#### 数据保存

- 针对微博：
	- 发帖人ID
	- 微博ID
	- 微博发布时间
	- 微博内容
	- 转发数、评论数、点赞数
	- 原微博ID(如果是转发微博)

- 针对评论：
	- 评论人ID
	- 评论ID
	- 评论内容
	- 被评论的微博ID
	- 赞数
	- 被回复的评论ID(如果是回复别人评论)

- 针对用户：
	- 用户ID
	- 昵称
	- 关注数
	- 被关注数
	- 微博数
	- 简介
	- 微博等级
	- 认证信息(如果加V)
	- 该用户相关微博是否爬完了


#### 数据清洗注意事项

- “转发微博”等无意义的可以直接过滤
- 图片忽略（表情不忽略）
- 抓取微博时，自动保存用户信息（用户信息可以选有价值的慢慢补）
